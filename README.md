# Old Photos Restore

Simple, trustworthy one-page website for a local family photo restoration and digitising service in South Wales (Caerphilly - Cardiff).

## Project goals
- Build a calm, clear, single-page website focused on trust and personal contact.
- Avoid unnecessary complexity (no React, no heavy JavaScript).
- Make the website readable for people aged 40-70+.

## Audience
- People aged 40-70+
- Small family photo archives (typically 5-50 prints)
- Concerned about originals and privacy
- Prefer personal contact over automation

## Scope (MVP)
- One-page layout:
  - Hero (what it is + where)
  - Services (scan, restore, return)
  - How it works
  - Examples (before/after)
  - Pricing (simple ranges)
  - Contact (phone, email, WhatsApp)
- Fast loading, accessible typography
- UK English copy

## Design choices
- Typography:
  - Headings - Merriweather
  - Body - Source Sans 3
- Tone:
  - Calm, clear, ethical
  - No marketing hype

## Tech stack
- HTML
- CSS
- Minimal JavaScript (only if required)

## Getting started
Open `index.html` in your browser.
